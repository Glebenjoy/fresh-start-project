# App Flow & User Journey: Visura AI

## 1. Screen: Home / Input (Вход)
* **UI:** Глубокий градиентный фон. Большая пульсирующая кнопка "Eye/Mic".
* **Action:** Ввод текста/голоса -> Кнопка "Visualize".
* **Logic:** При нажатии генерируем `session_id` и сохраняем текст в `localStorage`.

## 2. Screen: Processing (Умное ожидание)
**Цель:** Удержание внимания во время генерации (20-30 сек).
* **Visual:** Абстрактная "дышащая" сфера (Amber Glow).
* **Engagement:** Ротатор фактов (сменяются каждые 4-5 сек):
    * *"Знаете ли вы? 12% людей видят черно-белые сны..."*
    * *"Сны помогают мозгу обрабатывать эмоции..."*
    * *"Самый частый кошмар — падение..."*
* **Status:** "Синхронизация..." -> "Генерация образа...".

## 3. Screen: Revelation (Результат)
**Цель:** Вау-эффект и Конверсия.

* **Layer 1: Visual (100% Clarity)**
    * **HD Image:** Картинка видна четко.
    * **Share Button:** Генерирует **Брендированную карточку** (Картинка + Лого Visura + Цитата) для Instagram Stories.

* **Layer 2: Meaning (The Hook)**
    * **Headline:** "Тревожный сигнал подсознания" (Пример).
    * **Preview Text:** "Этот образ говорит о том, что вы подавляете гнев..." (Провокационный хук).
    * **Blur Mask:** Остальной текст скрыт.
    * **Unlock Button:** "Unlock Full Analysis (Free)".

## 4. Modal: Registration (Конверсия)
* **Trigger:** Нажатие на Unlock.
* **Message:** "Сохраните этот инсайт навсегда".
* **Auth:** Google / Email Magic Link.
* **Success:** Модалка закрывается -> Текст становится четким -> Сон сохранен в БД.

## 5. Screen: Journal (Дневник)
* Галерея прошлых снов (Masonry layout).
* Фильтр по эмоциям.
